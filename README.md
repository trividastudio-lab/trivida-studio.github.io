# ✅ Study Allowance App (공부 용돈 앱)

## 📚 앱 소개

**Study Allowance App**은 학생들의 학습 동기 부여를 위한 용돈 관리 앱입니다. 매일 정해진 과목들을 완료하면 용돈을 받을 수 있는 시스템으로, 체계적인 학습 습관 형성과 목표 달성을 도와줍니다.

### 🎯 주요 목적
- **학습 습관 형성**: 매일 정해진 과목들을 완료하는 규칙적인 학습 패턴 구축
- **동기 부여**: 용돈이라는 보상을 통한 지속적인 학습 의욕 증진
- **목표 관리**: 월간 목표 설정과 진행률 추적으로 성취감 제공
- **자기주도 학습**: 학생 스스로 학습 계획을 세우고 관리할 수 있는 환경 조성

### 👥 타겟 사용자
- 초등학생부터 고등학생까지의 학생
- 자녀의 학습 관리를 돕고자 하는 부모
- 체계적인 학습 습관을 만들고 싶은 학습자

### ✨ 핵심 기능
- **온보딩 시스템**: 국가/언어 로캘 기반 온보딩 프로세스 (언어 선택, 환영 가이드, 이름 입력, 국가별 초기 설정, 과목 설정, 목표 설정, 완료)
- **과목 관리**: 최대 5개 과목 설정 및 개별 용돈 금액 지정
- **일일 학습 체크**: 과목별 완료 상태 토글 및 실시간 피드백 (토스트 메시지)
- **월간 목표**: 달력 연계형 목표 일수 설정 및 진행률 추적 (목표 달성 시 황금 배지 및 폭죽 효과)
- **추가 용돈**: 특별한 성취나 보너스에 대한 추가 용돈 기록 (일일 최대 5개)
- **달력 뷰**: 월별 학습 현황을 한눈에 확인할 수 있는 시각적 달력 (진행률 바, 보너스 표시)
- **글로벌 로캘 지원**: 42개 국가, 45개 로캘(국가-언어 조합) 지원
- **데이터 관리**: 백업, 복원, 초기화 기능 (네이티브/웹 파일 시스템 지원)
- **PWA 설치 지원**: 매니페스트 기반 설치 가능 (서비스 워커 자동 등록은 현재 비활성)
- **Capacitor 통합**: 네이티브 앱 기능 (StatusBar, InAppReview, Filesystem, App, Share, FilePicker 등)
- **이름 편집**: 헤더에서 이름 클릭 시 직접 편집 가능
- **구글플레이 스토어 평점**: In-App Review 기능 지원

## 🏗️ 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 모던 UI/UX
- **JavaScript (ES6+)**: 모듈 시스템, 비동기 처리
- **PWA 리소스**: Web App Manifest, Service Worker 파일 포함

### 네이티브 앱
- **Capacitor**: 크로스 플랫폼 네이티브 앱 프레임워크
- **Capacitor 플러그인**:
  - StatusBar: 상태바 스타일 제어
  - InAppReview: 인앱 리뷰
  - Filesystem: 파일 시스템 접근
  - App: 앱 생명주기 관리
  - Share / FilePicker: 파일 공유 및 파일 선택

### 아키텍처
- **모듈화**: 기능별 파일 분리 (ES6 Modules)
- **상태 관리**: 중앙 집중식 state 관리
- **i18n 시스템**: 다국어 지원을 위한 자체 i18n 구현
- **로컬 저장소**: localStorage 기반 데이터 영구 저장
- **키보드 감지**: Visual Viewport API를 사용한 키보드 상태 감지
- **타이머/몰입 모드**: 과목별 타이머, Android 몰입 모드 연동

## 📱 주요 기능 상세

### 1. 온보딩 시스템
- **1단계: 언어 선택**: 브라우저 언어 자동 감지 및 국가-언어 로캘 선택
- **2단계: 환영 가이드**: 부모와 학생을 위한 앱 소개 및 안내 메시지
- **3단계: 이름 입력**: 개인화된 경험을 위한 이름 설정 (언어별 글자 수 제한)
- **4단계: 국가별 초기 설정**: 언어에 맞는 통화, 기본 과목 자동 설정 (백그라운드 처리)
- **5단계: 과목 설정**: 최대 5개 과목 설정 및 개별 용돈 금액 지정
- **6단계: 이달의 목표 설정**: 달력 연계형 목표 일수 및 목표 달성 보너스 설정
- **8단계: 완료**: 온보딩 완료 및 앱 시작

### 2. 메인 대시보드
- **헤더**: 사용자 이름, 앱 제목, 설정/달력 버튼
- **요약 섹션**: 
  - 이번 달 총 용돈
  - 목표 달성률 (진행률 바)
  - 목표 달성 시 황금 배지 표시
- **날짜 탐색**: 이전/다음 날짜 이동
- **과목 그리드**: 
  - 최대 5개 과목 표시
  - 완료/미완료 상태 토글
  - 완료 시 응원 토스트 메시지
- **추가 용돈**: 
  - 사유와 금액 입력
  - 일일 최대 5개 제한
  - 통화별 최대 금액 제한

### 3. 달력 뷰
- **월별 달력**: 학습 현황을 한눈에 확인
- **진행률 표시**: 각 날짜별 과목 완료 진행률 바
- **보너스 표시**: 추가 용돈이 있는 날짜 표시
- **날짜 클릭**: 해당 날짜로 메인 대시보드 이동

### 4. 설정 화면
- **과목 설정**: 
  - 과목명 및 금액 설정
  - 최소 1개, 최대 5개
  - 언어별 글자 수 제한
  - 통화 기호 자동 표시
- **이달의 목표**: 
  - 목표 일수 (15일 ~ 말일, 달력 연계형 자동 계산)
  - 목표 달성 보너스 금액
  - restConstant 기반 자동 계산 (총 일수 - restConstant = 목표 일수)
- **구글플레이 스토어 평점**: 
  - In-App Review 기능 지원 (네이티브)
  - 웹에서는 구글플레이 스토어 링크로 이동
- **데이터 관리**: 
  - 데이터 내보내기 (JSON, 네이티브/웹 파일 시스템 지원)
  - 데이터 가져오기 (네이티브: Documents 폴더에서 선택, 웹: 파일 선택)
  - 전체 데이터 초기화
- **저장 확인**: 변경사항이 있을 경우 저장 확인 모달 표시

### 5. 데이터 관리
- **로컬 저장**: 모든 데이터는 브라우저 localStorage에 저장
- **백업/복원**: JSON 파일로 데이터 내보내기/가져오기
  - 네이티브: Capacitor Filesystem 플러그인 사용 (Documents 폴더)
  - 웹: Web Share API 또는 다운로드
- **자동 저장**: 상태 변경 시 자동 저장
- **스키마 버전 관리**: 데이터 구조 변경 시 마이그레이션 지원
- **달별 데이터 관리**: 각 달의 과목 설정 및 목표를 독립적으로 관리

## 🚀 시작하기

### 설치 및 실행

1. **저장소 클론 또는 다운로드**
   ```bash
   git clone <repository-url>
   cd study-allowance-app
   ```

2. **로컬 서버 실행** (선택사항)
   ```bash
   cd www
   
   # Python 3
   python -m http.server 8000
   
   # Node.js (http-server)
   npx http-server
   
   # VS Code Live Server 확장 사용
   ```

3. **브라우저에서 열기**
   - `www/index.html` 파일을 브라우저에서 직접 열거나
   - 로컬 서버 주소로 접속 (예: `http://localhost:8000`)

### PWA 설치

1. 브라우저에서 앱 접속
2. 주소창의 설치 아이콘 클릭 또는
3. 브라우저 메뉴에서 "앱 설치" 선택

> 참고: 현재 `index.html`에서 서비스 워커 자동 등록 코드는 주석 처리되어 있습니다.

## 🔧 주요 기능 상세

### 과목 관리
- **최대 5개 과목**: 과목 추가/삭제/수정 가능
- **개별 금액 설정**: 각 과목별 용돈 금액 지정
- **언어별 글자 수 제한**: 
  - 한국어: 이름 7자, 과목명 15자
  - 영어: 이름 20자, 과목명 25자
  - 기타 언어별 제한 적용

### 목표 달성 시스템
- **월간 목표 일수**: 15일 ~ 해당 월 말일 (달력 연계형 자동 계산)
- **restConstant**: 각 달의 총 일수에서 빼는 상수 (기본값: 6일)
- **목표 달성 보너스**: 목표 달성 시 추가 보너스 지급
- **진행률 추적**: 실시간 진행률 바 및 퍼센트 표시 (25%, 50%, 75%, 90%, 100% 구간별 색상)
- **달성 시 특별 UI**: 
  - 황금 배지 및 축하 효과
  - 목표 달성 배지 클릭 시 폭죽 효과 (이스터 에그)
  - 진행률 바 숨김 및 황금빛 배너 표시

### 추가 용돈
- **일일 제한**: 하루 최대 5개
- **통화별 제한**: 각 통화별 최대 금액 제한
- **사유 입력**: 언어별 최대 글자 수 제한
- **실시간 합계**: 월간 총합 즉시 반영

### 달력 뷰
- **월별 표시**: 월 단위로 학습 현황 확인
- **진행률 바**: 각 날짜별 과목 완료 진행률 (하단 색상 바)
- **보너스 표시**: 추가 용돈이 있는 날짜에 빨간 점 표시
- **완료 표시**: 모든 과목 완료 시 날짜 셀에 completed 클래스 추가
- **오늘 표시**: 오늘 날짜에 today 클래스 추가
- **날짜 클릭**: 해당 날짜로 대시보드 이동
- **달력 안내**: 진행률 바, 완료 표시, 보너스 표시에 대한 안내 텍스트

## 🌐 다국어 지원 (i18n)

### 사용 방법
모든 텍스트는 `languages.js`의 i18n 시스템을 통해 관리됩니다:

```javascript
import { i18n } from './languages.js';

// 언어 설정
i18n.setLocale('ko');

// 텍스트 가져오기
const text = i18n.t('dashboard.header.user', { name: '홍길동' });
```

### 국가별 로캘 기본 설정
각 로캘은 다음 정보를 포함합니다:
- **통화**: 통화 코드, 기호, 소수점 자릿수
- **기본 과목**: 국가별 교육 과정에 맞는 기본 과목
- **기본 보너스**: 국가별 통화에 맞는 기본 보너스 금액
- **최대 금액**: 통화별 최대 입력 금액

- **Filesystem**: 네이티브 파일 시스템 접근 (데이터 백업/복원)
- **App**: 앱 생명주기 관리, 뒤로가기 버튼 처리
- **Share / FilePicker**: 파일 공유 및 파일 선택

## ⚠️ 주의사항

### 데이터 관리
- **백업 권장**: 중요한 데이터는 정기적으로 내보내기로 백업
- **초기화 주의**: 데이터 초기화는 복구 불가능
- **브라우저 데이터**: 브라우저 데이터 삭제 시 모든 데이터 손실
- **달별 설정**: 각 달의 과목 설정을 변경하면 해당 달의 기록이 초기화됨

### 브라우저 호환성
- **최신 브라우저 권장**: Chrome, Firefox, Safari, Edge 최신 버전
- **localStorage 필수**: localStorage를 지원하는 브라우저 필요
- **ES6+ 모듈**: ES6 모듈 시스템을 지원하는 브라우저 필요
- **Visual Viewport API**: 키보드 감지를 위해 Visual Viewport API 지원 브라우저 권장

### 네이티브 앱
- **Android**: Android 5.0 (API 21) 이상 권장
- **플러그인 동기화**: 플러그인 변경 시 `npx cap sync android` 실행 권장
